# wp2bs-theme-starter

## (wp2 = wordpress + webpack) + (bs = bootstrap-sass) theme starter

Heavily inspired:

https://github.com/MozaikAgency/wp-theme-starter

### how to use

edit ./config.js & added your php files into ./theme directory, and..

```bash
### Install deps, first
npm i

### first build & run browsersync dev server
gulp build && gulp
```
